 Task CLI Project

This is a **Command-Line Interface (CLI) Task Tracker** built with Python.  
It allows you to **add, update, delete, and manage tasks** directly from the terminal, and stores your tasks in a JSON file.

 **Features**
- Add new tasks with multi-word descriptions  
- Update existing tasks  
- Delete tasks  
- Mark tasks as **in-progress** or **done**  
- List all tasks with status (`todo`, `in-progress`, `done`)  
- Timestamps for task creation and updates (`createdAt`, `updatedAt`)  
- Simple CLI interface with help menu
